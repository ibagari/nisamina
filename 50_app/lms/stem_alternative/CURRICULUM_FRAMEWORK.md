## Pillar 1 — Mathematics (Singapore CPA + Bar Modeling)

Per F-065-AMENDMENT: Singapore Math is the gold-standard PISA/TIMSS leader. CPA = **Concrete → Pictorial → Abstract** progression. Bar Modeling = visual problem-solving scaffold for word problems.

### Per-band rollout

| Band | Approach | Garifuna concrete anchors |
|---|---|---|
| 01_PreK | Concrete only | cassava-tubers + canoe-models + dabuyaba-drumheads for counting/shapes |
| 02_K | Concrete + early Pictorial | shells + woven-strips for sets + comparisons |
| 03_G1-G2 | C → P transition; bar models introduced | community-marketplace problems; sere-fish-counting |
| 04_G3-G5 | Pictorial-dominant; abstract emerging | hudutu-recipe-ratio problems; canoe-distance-time |
| 05_G6-G9 | Abstract algebra; bar models for fractions/percent | ereba-batch-yield problems; tidal-distance |
| 06_G10-G12 | Abstract; AOPS + IGCSE + Cambridge level | Caribbean-fisheries economic-modeling |
| 07_UnivPrep | College-prep + AOPS competition; AP-equivalent | Garifuna ethnomathematics scholarship |

**Source corpus:** Spectrum series (G3-G5) + Singapore Math (cross-band) + Bar Modeling exemplars + AOPS (G6-12) + Cambridge IGCSE/A-Level (G10-12) + 180-Days-of-Math (G3-G5).

## Pillar 2 — Science (Inquiry-based + Caribbean-ecological)

Per F-065-AMENDMENT: PISA-leader pedagogy = inquiry-driven, hands-on investigation, with cultural-ecological anchoring.

### Per-band rollout

| Band | Approach | Caribbean anchors |
|---|---|---|
| 01_PreK | Sensory exploration | shells + sand + water + plant matter |
| 02_K | Observation + classification | local fauna (iguanas, parrots) + flora (cassava, plantain) |
| 03_G1-G2 | Question-based exploration | tide pools + reef ecology |
| 04_G3-G5 | Inquiry cycles (question → hypothesis → test) | mangrove ecosystems + canoe-buoyancy + cassava-fermentation |
| 05_G6-G9 | Lab-based + data analysis | hurricane patterns + coral bleaching + traditional ecological knowledge |
| 06_G10-G12 | Discipline-specific (Bio/Chem/Phys/Earth) | Garifuna ethnobotany + Caribbean marine biology |
| 07_UnivPrep | College-prep with research methods | regional climate science + indigenous knowledge integration |

**Source corpus:** Cambridge University Science full textbook + 379 K-12 Science files spanning all 7 bands.
